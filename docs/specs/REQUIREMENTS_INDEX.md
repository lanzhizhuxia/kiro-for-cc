## 📝 使用说明

1. **创建新需求前**: 查看"下一个可用编号"
2. **分配编号后**: 在表格新增一行，标记为"📋 Pending"
3. **状态变更时**: 更新对应行的状态和日期
4. **编号不回收**: 取消的需求编号也保留在索引中

## 📊 状态说明

- 📋 **Pending**: 待处理（在 `docs/specs/pending/`）
- 🚧 **In Progress**: 进行中（在 `docs/specs/in-progress/`）
- ✅ **Completed**: 已完成（在 `docs/specs/completed/`）
- ❌ **Cancelled**: 已取消
